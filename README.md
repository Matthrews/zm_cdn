# zm_cdn

A light CND center for personal development

based on jsDelivr and Github

## 如何使用

上传一张图`cammader.png`到`images`目录，则最终图片访问路径为：
https://cdn.jsdelivr.net/gh/Matthrews/zm_cdn/images/cammader.png
